# Zomato-Sales-Analysis
The Zomato Sales Analysis project focused on analyzing restaurant and customer-related data to identify trends in cuisines, ratings, delivery services, and pricing across multiple cities and countries.
I used SQL for data extraction and analysis. I performed joins, aggregations, filtering, and trend analysis to uncover insights from the dataset.

Some of the major analyses included:

Top-rated cuisines
Restaurant distribution across cities
Online delivery availability
Cost segmentation
Ratings analysis
Service availability trends

After analysis, I created dashboards in Tableau and Power BI to visualize:

Restaurant growth trends
Country-wise and city-wise analysis
Price range segmentation
Online delivery insights
Rating distributions

The project helped demonstrate how data can support business expansion decisions and improve customer engagement strategies.
